Задача №1

Почему код даёт именно такие результаты?

var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2
d = b++; alert(d); // 1
c = (2+ ++a); alert(c); // 5
d = (2+ b++); alert(d); // 4
alert(a); // 3
alert(b); // 3

1. c = ++a; alert(c); // 2
2. d = b++; alert(d); // 1
Одной из наиболее частых числовых операций является увеличение или уменьшение на единицу.
Для этого существуют даже специальные операторы.
Инкремент++ увеличивает переменную на 1.
Когда оператор идёт после переменной — это «постфиксная форма» или «постфикс». «Префиксная форма» или «префикс» — это когда оператор идёт перед переменной.
Префиксная форма ++a увеличивает a и возвращает новое значение 2. Gостфиксная форма b++ также увеличивает b, но возвращает старое значение (которое было до увеличения). Результат покажет 1.

3. c = (2+ ++a); alert(c); // 5
Сначала выполняется инкремент префикс, увеличение a=a+1, а потом уже с новым значением "a" производятся другие операции т.е. 2 + ((a=2)+1) = 5

4. d = (2+ b++); alert(d); // 4
В данном случае выполняется операция инкремент постфикс b=b+1, но сложение происходит с предыдущим значением "b", т.е. 2 + 2 = 4.

5. alert(a); // 3
Значение "a" увеличивалось в предыдущих операциях инкремента 2 раза на 1, т.е. a=(a=1)+1+1

6. alert(b); // 3
Значение "b" увеличивалось в предыдущих операциях инкремента 2 раза на 1, т.е. b=(b=1)+1+1

Задача №2

Чему будет равен x?
var a = 2;
var x = 1 + (a *= 2);

x = 5, т.к. операция в скобках (умножение) первична, её приоритет выше, далее операции за скодками (сложение),т.е. x = 1 + (a = a * 2 = 4) = 5.
